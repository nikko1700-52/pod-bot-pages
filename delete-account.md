# Suppression des Données Utilisateur

## Comment supprimer vos données

Pour demander la suppression complète de vos données de notre application:

1. Envoyez un email à: **nico25850@gmail.com**
2. Indiquez: "Suppression de données POD-BOT"
3. Fournissez votre ID utilisateur ou email utilisé

**Délai de traitement:** 7-30 jours

Nous supprimerons:
- Tous les logs d'utilisation
- Les données de session
- Tout historique d'interaction
- Les données de configuration

Pour plus d'infos sur notre politique: https://nikko1700-52.github.io/pod-bot-pages/privacy.md
